## 基本绘图概念
- Figure 与 Axes
- 坐标系统
- Artist 对象层次
- 图层与 Z-order
- 示例：
```python
# 基本绘图示例
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = np.sin(x)

# 创建图表
fig, ax = plt.subplots()
ax.plot(x, y)
ax.set_title('正弦波')
ax.set_xlabel('X 轴')
ax.set_ylabel('Y 轴')
plt.show()
```
