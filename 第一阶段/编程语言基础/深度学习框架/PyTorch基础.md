- 张量创建与初始化
- 张量操作与运算
- 形状操作
- 索引与切片
- GPU 计算
- 示例：
```python
# 张量操作基础示例
import torch

# 创建张量
x = torch.tensor([[1, 2], [3, 4]], dtype=torch.float32)
y = torch.ones(2, 2)

# 张量运算
z = x + y
print(z)

# GPU 运算 (如果可用)
if torch.cuda.is_available():
    device = torch.device("cuda")
    x_gpu = x.to(device)
    y_gpu = y.to(device)
    z_gpu = x_gpu + y_gpu
    print(z_gpu)
```
