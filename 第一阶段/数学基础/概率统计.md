- 全概率公式
- 示例：
```python
# 概率计算示例
import numpy as np
from scipy import stats

# 模拟投掷硬币
n_trials = 10000
coin_flips = np.random.choice(['H', 'T'], size=n_trials, p=[0.5, 0.5])

# 计算正面朝上的概率
p_heads = np.mean(coin_flips == 'H')
print(f"正面朝上概率: {p_heads:.4f}")
```
